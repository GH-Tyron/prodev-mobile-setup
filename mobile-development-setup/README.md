## Objective

Set up **Expo Go** on a mobile device to enable rapid testing and development of React Native applications using Expo.

---

## Steps to Install Expo Go

### 1. Visit Expo Go Homepage

Open the official Expo Go website:

* [https://expo.dev/go](https://expo.dev/go)

### 2. Select the Latest SDK Version

On the homepage, ensure you are viewing or selecting the **latest Expo SDK version** to maintain compatibility with current Expo projects.

### 3. Install Expo Go on Your Device

Choose the appropriate platform:

* **Android**

  * Click **Install** to open the Google Play Store
  * Search for **Expo Go**
  * Tap **Install**

* **iOS**

  * Click **Install** to open the Apple App Store
  * Search for **Expo Go**
  * Tap **Get** and authenticate if required

### 4. Open the Expo Go App

After installation:

* Launch **Expo Go** on your mobile device

### 5. Create or Log In to an Expo Account

* Sign up for a new Expo account if you do not have one
* Log in if you already have an account

This allows you to sync projects and scan QR codes generated by the Expo CLI.

---

## Verification

* Expo Go opens successfully
* User is logged in
* App is ready to scan QR codes from `expo start`
